# **App Name**: Ryan Wez

## Core Features:

- Hero Section: Display a hero section with a brief personal introduction.
- About Me Section: Showcase an 'About Me' section detailing skills, experience, and passions.
- Dynamic Projects Section: Dynamically display projects fetched from the backend API, with title, description, technologies used, and links to live demo and source code.
- Skills/Technologies Section: List technical skills and technologies in a dedicated section.
- Contact/Social Links: Include contact and social media links for easy access.
- Admin Panel: Enable administrators to add, edit, and delete projects via a password-protected admin panel.

## Style Guidelines:

- Primary color: A subdued blue (#5DADE2), evoking a sense of trust and professionalism.
- Background color: Light gray (#F0F4F8), offering a clean, modern backdrop that highlights content without distraction. This color is of a similar hue to the primary, desaturated and lightened for use as a background.
- Accent color: A complementary shade of purple (#A98DDA). The brightness is tweaked so it can be used to provide emphasis on interactive elements such as buttons and links.
- Headline font: 'Space Grotesk' sans-serif for a modern, tech-oriented aesthetic. Body font: 'Inter' sans-serif for readability and clean design.
- Implement a clean, minimalist design inspired by ryanwez.github.io, focusing on readability and easy navigation.
- Utilize a responsive grid system for project display, ensuring optimal viewing on all devices.
